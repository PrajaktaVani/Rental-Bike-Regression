# Rental-Bike-Regression
This project aims to predict the number of bikes rented per hour/day based on various environmental and seasonal factors. The goal is to help rental companies optimize bike availability, reduce shortage/surplus, and improve customer experience.
Objectives
Understand patterns in weather, seasonality, and user behavior
Clean and prepare a real-world dataset
Train and evaluate regression models
Predict future rental demand
